Usos de tabs

Esta sección tiene algunos códigos para el uso de los Tabs

Marcado HTML

Lo recomendable es que la lista de navegación de haga con una lista ordenada ul que tenga la clase class="nav nav-tabs" y el rol role="tablist", los elementos de la lista serán li con el rol role="presentation" esto para garantizar la accesibilidad del componente. Finalmente dentro de cada li debe ir un elemento a con el atributo data-toggle="tab".

Código inline

Para navegar entre los tabs sin uso de jQuery deben añadir la propiedad data-toggle al vínculo y en href seleccionar el id del tab tal como se muestra en el ejemplo.

a data-toggle="tab" href="#sectionB"

jQuery

El siguiente código permite mostrar por medio de una acción de jQuery un tab específico, en #id-tab deben poner el id que ustedes den al tab.

$('#tabs-content a[href="#id-tab"]').tab('show'); // Muestra el tab con el ID respectivo
$('#tabs-content a:first').tab('show') // Muestra el primer tab
$('#tabs-content a:last').tab('show') // Muestra el último tab
$('#tabs-content li:eq(2) a').tab('show') // Selecciona el tercer tab (el index inicia en 0)

Lo ideal sería es que si el usuario crea una marca nueva al volver lo recomendable sería que lo dejara de inmediato en el tab de marcas.
